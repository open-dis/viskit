Viskit Run Simulation State Machine

Run Simulation States

       Ready. Simulation initialization complete, ready to begin first replication.

Run - Resume.

  • Run:       begin simulation replications in continuous mode until completion.
  • Resume: perform remaining simulation replications in continuous mode until completion.

Pause - Step.

  • Pause: pause continuous replications, begin single-step mode, await Resume or Step or Stop.
  • Step:    perform one simulation loop within current replication, continue single-step mode, await Resume or Step or Stop.

       Done. Simulation replications are completed or stopped, ready to prepare Analyst Report.

Additional VCR Buttons

Clear. Clear all text in console area.

Stop. Halt simulation replications, set state Done, ready to prepare Analyst Report.
