Assembly Simulation State Machine

This state machine describes how Viskit VCR buttons work when running an Assembly Simulation.

Simulation States

       INITIALIZE. Perform simulation initialization, ready to begin first replication.

RUN

  • Run mode:       begin simulation replications in continuous mode until completion.
  • Resume mode: continue running remaining simulation replications in continuous mode until completion.

PAUSE

  • Pause mode: pause continuous simulation replications, begin single-step mode, await Resume or Step or Stop.
  • Step mode:    perform one simulation loop within current replication, continue single-step mode, await Resume or Step or Stop.

       DONE. Simulation replications are completed or stopped, ready to review Analyst Report.

Additional VCR Buttons

Clear. Clear all text in console area.

Stop. Halt simulation replications, set state Done, ready to prepare Analyst Report.

Repeating a Simulation

Select Assembly Editor tab, once again press button Initialize Assembly for Simulation Run.
