Visual Simkit

Visual Simkit (Viskit) is a graphical application for defining Discrete Event Simulation (DES) models. Event graph and assembly models are used to autogenerate executable Java source code using the Simkit library.

           

The Viskit application allows analysts to build a collection of event graphs that describe the behavior of various entities in a simulation environment.

Visual Simkit is under active development. Reports and feedback to viskit@movesInstitute.org are welcome.
